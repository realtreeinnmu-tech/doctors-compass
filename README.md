# Simple Alarm

すごくシンプルな UI のブラウザ向けアラームアプリです。HTML / CSS / JavaScript のみで動作し、ビルドも依存関係もありません。

## 特徴

- 大きく表示される現在時刻
- 時刻を選んで「アラームを追加」するだけのシンプル操作
- 各アラームを ON / OFF トグル、不要なら × で削除
- 設定時刻になるとビープ音を鳴らし、ブラウザ通知も発火
- アラームは LocalStorage に保存されるためリロードしても消えない
- ライト / ダークモードに自動対応

## 使い方

1. リポジトリをクローン、または zip でダウンロード
2. `index.html` をブラウザで直接開く

   または、簡易サーバ経由で開きたい場合:

   ```sh
   python3 -m http.server 8080
   # → http://localhost:8080
   ```

3. ブラウザで通知を許可するとアラーム時に通知も表示されます

> 音声を鳴らすために、初回はページ上で何かをクリックする必要があるブラウザがあります(自動再生制限)。

## ファイル構成

- `index.html` … マークアップ
- `style.css` … スタイル
- `script.js` … 時計・アラーム管理・サウンド
